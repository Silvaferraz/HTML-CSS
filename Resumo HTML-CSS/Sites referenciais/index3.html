Imagens

Para usar imagens que estão na mesma pasta basta digitar a tag "img" + Enter pra carregar o código automático, e dentro do parâmetro "src"(source/origem) digitar o nome dado a imagem ou segurar "Ctrl + Backspace" que ele irá mostrar os arquivos disponíveis na pasta, e o parâmetro "alt" serve para descrever a imagem caso ela não carregue.

  • Tomar muito cuidado com imagens com direitos autorais, pois podem comprometer o seu site, sites como unsplash.com e pexels.com podem ser usados para procurar imagens que possam ser usadas sem problemas, porém tomar cuidado mesmo assim pois podem ter imagens com direito autoral

Favicon

Os favicon são ícones que aparecem na aba lá em cima, no navegador, como o emoji 💩 que você vê lá em cima nesse site, os favicons podem ser de vários tipos, imagens, texto ou emojis, arquivos favicon possuem a extensão ".ico"

Para add um favicon deve-se escrever acima de <title> a tag "link:favicon" e dentro de "href", carregar o favicon.
