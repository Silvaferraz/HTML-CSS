Domínios e Hospedagens

Para ter uma DNS é preciso um domínio e uma hospedagem para que outras pessoas também possam acessar a sua URL

Domínio é o nome que identifica o site, nome único, normalmente nas plataformas de domínios elas tem um pagamento anual

Hospedagem é o local onde meu site vai estar armazenado, normalmente nas plataformas é pago mensalmente

URL

  • www é o subdomínio principal do servidor web
  • /xxxxx é o caminho da URL

TLD

Top Level Domain é a parte do endereço que vem logo após o último ponto da URL, e identifica a finalidade do site, por exemplo, ".com" trata de sites comerciais, ".gov" trata de sites governamentais

  • gTLD → Generic top level domain, são os mais conhecidos e utilizados, .org, .net, .info etc.
  • sTLD → Sponsored top level domain são aqueles patrocinados por entidades, empresas e governos como, .gov, .edu, .travel etc.
  • ccTLD → Country code top level domain são para identificar sites de países específicos como, .br, .uk etc.
