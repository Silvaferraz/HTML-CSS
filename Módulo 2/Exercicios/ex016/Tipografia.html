Anatomima do tipo

Toda letra e toda fonte pra ser construída precisa ser estudada

Cada elemento da letra possui "classificações",

  • Altura x: é o tamanho base de toda letra minúscula que tem na sua base o tamanho da letra x em minúsculo.
  • Altura das maiusculas: é a altura que a letra maiuscula ocupa a partir da base da altura x, até a parte final do vertical da letra.
  • Ascendente: é a altura base x mais oq é vazado verticalmente pra cima em letras maiusculas e minúsculas.
  • Descendentes: é a altura base x mais oq é vazado verticalmente pra baixo em letras maiusculas e minúsculas.
  • Corpo: é a altura total somando todas as outras partes da letra e é ele que nós configuramos em CSS

  • Serifas: Tracinhos que algumas fontes possuem nas letras, que indiretamente influenciam no nosso cerebro na hora da leitura, pois não lemos letra por letra e sim a palavra como um todo.
  • glifos=letras
  • Cojunto de todos os glifos de uma determinada família = Fonte
  • Dentro de cada fonte existem famílias ou formatos/variações diferentes de uma mesma fonte.

    • Categorias de Fontes:

  • Serifadas = tem serifa
  • sans-serif=sem serifa
  • Monoespaçada=pode ser c/ ou s/ serifa, todas letras ocupam o mesmo espaço
  • handwriting=scriptada, fonte simulada pra parecer escrita a mão
  • display=não tem nenhuma caracteristica básica, não tem "regras" fixas, se baseiam em ideias pra lembrar filmes e etc.
